# Project Overview
This project analyzes IPL matches, players, and team statistics to provide actionable insights for fans, analysts, and teams. It includes interactive dashboards, data visualizations, and detailed analytics.
Key Objectives:
- **Track team performance over seasons**
- **Identify top-performing players (batsmen, bowlers)**
- **Analyze match outcomes, winning patterns, and venue impact**
- **Explore player statistics like strike rates, economy rates, and averages**
- **Visualize trends in runs, wickets, and boundaries**
________________________________________
# FeatureS
- **Interactive Power BI Reports (`.pbix`)**
- **Cleaned IPL datasets (matches, players, teams, scores)**
- **Data transformations & DAX measures for advanced analytics**
- **KPI tracking: highest run-scorers, top bowlers, team win rates**
- **Charts & visuals: bar charts, line charts, scatter plots, heatmaps**
- **Predictive insights (optional): identify players/teams likely to perform in upcoming matches**
